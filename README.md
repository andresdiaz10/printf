# _printf
Recreation of the C standard function, printf.
# Dependencies
_printf will be compiled on Ubuntu 20.04 LTS using gcc 9.3.0.
# Usage
Example main.c:
```
#include "holberton.h"

int main(void)
{
  _printf("Hello, World!");
  return (0);
}
```
Compilation:
```
$gcc *.c -o out

```
Output:
```
$./out
Hello, World!
$
```
